# Contribuindo com copilot-openai-workshop

Este repositório aceita contribuições em código, documentação e tradução.

## Como contribuir

1. Fork o repositório em https://github.com/seu-usuario/copilot-openai-workshop
2. Crie um branch:

   ```bash
   git checkout -b feature/nome-da-contribuicao
   ```

3. Faça suas alterações (melhorias de código, novos laboratórios, exemplos em outra linguagem).
4. Adicione commits com mensagens claras.
5. Envie para o fork:

   ```bash
   git push origin feature/nome-da-contribuicao
   ```

6. Abra um **Pull Request** na branch principal `main`.

## Sugestões de contribuições

- Adicionar `lab-05` com um exemplo de **Agentes de IA** (por exemplo, usando LangChain).
- Criar exemplos em **JavaScript** ou **.NET**.
- Melhorar a documentação em português ou traduzir para inglês.
- Incluir exemplos de **segurança de dados** e **filtros de conteúdo**.

## Código de conduta

Ao contribuir, você concorda em seguir o [Código de Conduta da comunidade GitHub](https://docs.github.com/pt/site-policy/github-terms/github-community-code-of-conduct).
